## 介绍

- `packages` 下每个项目可以单独存到 `git` 仓库中，这里为了方便管理，统一放在 `packages` 目录下

- `sdk` 项目可以打包上传到 `npm`

## 安装依赖

```sh
cd ./frontend/web/

# 一定要用 pnpm
pnpm install
```
