## SDK

### api 模块

- `axios` 请求配置 `config`

- 常用请求 `getUserInfoApi`、`getRoutesApi`

### app 模块

- `env` 环境变量

- 其他通用配置

### client 模块

- `location`、`navigate`, 解决应用之间跳转问题

### components 模块

- 外部传入组件, 主要是获取路由组件

### hooks 模块

- `useRoot` 获取 `sdk` 信息

- `useMicroState` 获取 `子应用加载状态` 信息

- `useTheme` 获取 `主题` 信息

### router 模块

- 路由信息、菜单信息、子应用信息

### storage 模块

- 通用 localStorage 封装

### store 模块

- 全局状态管理

## ui 模块

- `ProLayout` 模块

## utils 模块

- 通用方法
